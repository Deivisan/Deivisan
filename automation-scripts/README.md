## 🔧 Como Usar

1. **Clone o repositório**
```bash
git clone https://github.com/deivisan/automation-scripts.git
cd automation-scripts
```

2. **Instale dependências**
```bash
pip install -r requirements.txt
```

3. **Execute um script**
```bash
python sistema/limpeza_temp.py
```
